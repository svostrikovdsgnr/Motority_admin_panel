# Motority Brand Admin — Prototype

Clickable UI prototype for the Motority brand admin platform.  
Built with vanilla HTML, CSS, and JS. No dependencies, no build step.

## Structure

```
motority-admin/
├── index.html   — markup & layout
├── styles.css   — design tokens, themes, component styles
├── app.js       — navigation, modals, theme switcher, interactions
└── README.md
```

## Deploy to GitHub Pages

1. Push all files to a GitHub repository
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch** → `main` → `/ (root)`
4. Your site will be live at `https://<username>.github.io/<repo>/`

## Features

- 10 admin sections: Dashboard, Landing Page Editor, Cars Catalog, Media Library, Brand Style, Community, Users & Roles, Notifications, Version History, Settings
- Dark / Light theme switcher
- Clickable modals, toggles, tabs, chips
- Live preview panel (Landing Page Editor)
- Toast notifications on all actions
